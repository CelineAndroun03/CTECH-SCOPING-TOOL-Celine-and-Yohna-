# CTECH-Effort-Estimation-Tool
This repository is destinated to the CTECH Effort Estimation Tool Model Development

## Setup
1. Clone the repo:
- git clone https://github.com/your-username/CTECH-Effort-Estimation-Tool.git
- cd CTECH-Effort-Estimation-Tool
2. Create a virtual environment
- python -m venv venv
- .\venv\Scripts\activate
3. Install requirements
- pip install -r requirements.txt

## Folder Structure
 - src/ - source code
 - data/ - datasets
 - tests/ - unit tests
